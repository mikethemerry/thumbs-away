# Privacy Policy

**Thumbs Away**  
**Effective Date:** December 2024

## Summary

Thumbs Away does not collect, store, transmit, or share any personal data or browsing information. Your privacy is fully protected.

---

## Information We Collect

**We do not collect any data.** The extension:

- Does not collect personal information
- Does not track your browsing activity
- Does not send any data to external servers
- Does not use analytics or tracking tools

## Data Storage

The extension stores only your toggle preferences (which platforms have feedback buttons hidden). This data:

- Is stored locally in your browser using Chrome's storage API
- Syncs across your browsers if you're signed into Chrome (via Google's sync)
- Contains only three boolean values (on/off for each platform)
- Can be cleared by uninstalling the extension

## Permissions

The extension requests these permissions:

| Permission | Purpose |
|------------|---------|
| `storage` | Save your on/off preferences for each platform |
| `activeTab` | Apply CSS changes to the current tab |
| `scripting` | Inject CSS to hide buttons |
| Host permissions | Access only claude.ai, chatgpt.com, chat.openai.com, and gemini.google.com |

## Third Parties

This extension does not share data with any third parties because it does not collect any data.

## Website Access

The extension only accesses these specific websites:
- claude.ai
- chat.openai.com
- chatgpt.com
- gemini.google.com

It cannot access any other websites.
